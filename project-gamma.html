Bombo Clat

An action-packed project featuring retro-inspired character mechanics and pixel-perfect animations.

About the Game

Bombo Clat is a fast-paced 2D platformer developed in Unity for a Game Jam. The project focuses on responsive character movement and a distinct retro aesthetic, created to test rapid prototyping skills under tight deadlines.

What I Did

• Game Jam Development: Engineered the core 2D platforming mechanics and character controller using Unity and C#.
• Animation & Art: Hand-painted character assets and designed the core "Idle" animation from scratch using Aseprite.
• Level Engineering: Swiftly designed and implemented level structures to showcase the game's agility-focused gameplay.
